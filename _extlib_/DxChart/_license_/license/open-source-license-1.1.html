DxChart 
DxChart Copyright

DxChart is a JavaScript charting and graphing library for websites. It was first created in August 2023 and aims to be a charting library optimized for nexacro. Growing to 50-60 different types of charts using HTML5 canvas, DxChart can present different types of data quickly and efficiently.

DxChart uses RGraph (https://www.rgraph.net/) as a reference model under a commercial license.

DxChart is suitable for any website using web-based or nexacro-based.
